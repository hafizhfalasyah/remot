## 📌 Overview

**ReMot (Rental Motor)** is a web-based motorcycle rental platform designed to streamline the booking and payment process through a fully digital experience.

The system enables users to reserve motorcycles in advance and complete secure online transactions, eliminating the need for on-site reservations and reducing uncertainty in vehicle availability.

This project demonstrates the implementation of modern web development practices using Laravel, combined with real-world payment gateway integration.

---

## 🎯 Key Highlights

- End-to-end online booking system  
- Integrated Midtrans payment gateway  
- Role-based authentication (Admin & User)  
- Scalable backend architecture using Laravel  
- Designed with a focus on usability and efficiency
